NullPointerException occurs when trying to use an object reference that has not been initialized.

IndexOutOfBoundsException happens when accessing an array or list index that is outside valid range.

ClassNotFoundException occurs when the JVM cannot locate a class definition at runtime.

SQLException may occur due to invalid queries, connection failures, or incorrect database configurations.

NumberFormatException occurs when attempting to convert a string to a numeric type but the string format is invalid.

IllegalArgumentException is thrown when a method receives an inappropriate argument.
