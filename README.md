## Introduction
The AI Java Debugger is a tool designed to help developers debug their Java applications using AI-driven insights and suggestions.

## Installation
To install the AI Java Debugger, follow these steps:
1. Ensure you have Java Development Kit (JDK) installed on your machine.
2. Clone the repository:
   ```bash
   git clone https://github.com/codejasleen/ai-java-debugger.git
   ```
3. Navigate to the project directory:
   ```bash
   cd ai-java-debugger
   ```
4. Compile the project:
   ```bash
   javac src/**/*.java
   ```
5. Run the debugger:
   ```bash
   java -cp bin Main
   ```

## Usage
To use the AI Java Debugger, simply run the command below after installation:
```bash
java -cp bin Main
```
